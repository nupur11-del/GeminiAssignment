DISPLAY

When we create an element to display on the page, it renders as a box & we have seen how to position those boxes on the page, but there’s more we can do with them. CSS’s Display property specifies the type of rendering box that is created from our HTML. Let’s check out some of the different display styles we can use. The display property specifies if/how an element is displayed. Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline

Block level element

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

Examples of block-level elements: div,li,section etc

This is block level element

Inline element

Element can start anywhere on an existing line. Height and width properties have no effect. Elements that are inline by default include span,input,button,img etc An inline element does not start on a new line and only takes up as much width as necessary.

This is inline element

Override element

Changing an inline element to a block element, or vice versa, can be useful for making the page look a specific way, and still follow the web standards. 4. Float

The CSS float property is a positioning property. It is used to push an element to the left or right, allowing other element to wrap around it. It is generally used with images and layouts.

  • none: The element does not float, (will be displayed just where it occurs in the text). This is default
  • left:The element floats to the left of its container
  • right:The element floats the right of its container
